является ли элемент строчным (inline)

Строчные элементы внутри родительского можно центрировать с помощью text-align: center

.parent-container { text-align: center; }

Блочный элемент

Блочный элемент можно центрировать, указав для margin значение auto

.container { margin: auto; }
